OBSERVATION 3 — Fault injected
--------------------------------
Time    : 2026-06-19 ~11:03:13 UTC
Where   : PowerShell terminal (local machine)
Tool    : fault-inject.ps1
What    : Script executed with 5-second countdown abort window.
          Two changes made to nsg-backend:
          (a) AllowAppToBackend moved from priority 200 → priority 300
          (b) New rule LabBlock8080 created: Deny / Icmp / priority 150 /
              source 10.10.1.0/24 / destination *
          Result: LabBlock8080 at priority 150 now intercepts ICMP before
          AllowAppToBackend at priority 300 can permit it.
          Azure NSG evaluates rules lowest-priority-number first, so Deny wins.
